# FormatMyDoc

FormatMyDoc converts messy text into clean, editable documents using AI.

## Tech Stack

Frontend
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- Tiptap

Backend
- Express
- TypeScript
- Groq API
- Tesseract.js
- Puppeteer

## Run Frontend

cd client
npm install
npm run dev

## Run Backend

cd server
npm install
npm run dev
